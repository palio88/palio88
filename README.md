## Selected Builds

### Human-Approved Job Application Bot

Built a local workflow to collect, score, tailor, and track fully remote job opportunities.

**What it does**
- Pulls roles from ATS and remote-job sources
- Scores opportunities against a structured career profile
- Generates role-specific application packets and tailored resume variants
- Tracks submitted, skipped, blocked, and manual applications in an Excel-ready tracker
- Retrieves scoped email verification codes from Apple Mail only for the active application

**Tools used**
Python, Playwright, AppleScript, Greenhouse, Lever, Ashby, JSON, CSV, XLSX, browser automation, human-in-the-loop approval controls.

### AI Trading Research And Execution Tooling

Built and iterated on private trading-system workflows for options/futures research, backtesting, reporting, and guarded live-ops decisions.

**What it does**
- Runs strategy reports and backtest comparisons
- Tracks live-safety checks before activation or restart decisions
- Produces evidence-backed summaries and report artifacts
- Uses explicit gates around route health, open positions, pending entries, and risk state

**Tools used**
Python, pandas-style data workflows, JSON/JSONL, brokerage/trading APIs, reporting scripts, Telegram/PDF delivery workflows, risk controls, automation logs.
